Shakle Promise Library
Shakle is a lightweight and educational Promise module hosted on npm.

This package provides a Promise class and some advanced helper methods all written in plain vanilla JavaScript. The module is meant to be used functionally, but Shakle is also meant to be used as an educational resource for those in the JavaScript community who want to be introduced, get better, or need a refresher to Promises.
The source code is heavily commented as to provide a deep dive into not only what Promises do but also how they do it.
Comprehensive Mocha tests are set up for the user to easily explore the module's functionality and even extend it's functionality using Test Drive Development.
Feel free to tinker!

Ripple
Ripple is a web service that allows users to easly integrate thier favorite web services with each other

With Ripple, your apps work for you, behind the scenes, even when you're not using them.
Ripple is all about chain reactions. When you post a photo in Instagram, Ripple will upload it to your Dropbox. When you finish a run using Fitbit, Ripple will record your run in Google Sheets.
In Ripple, a relationship between two apps is called a concoction. A concoction follows basic cause and effect logic. Select a trigger app and event, then choose your action app and event.

In addition to handling authentication and automatic access token renewal for tens of APIs, the app's performant backend is optimized to handle hundreds of concurent requests and listens in the background for webhook responses from all of the user's authorized APIs. Ripple's servers dynamicaly and efficently query users' concoction data to respond appopritly and accordingly.

Wanderer
Warderer is novel trip planning app. Just have it listen to a list of places you would like to visit and Wanderer Plans the optimal route between those destinations.

The app uses the browser's Webspeech API to transcribe the description of your trip, then custom algorithms parse this request and in turn use the Google Maps API to plan a trip that includes all the mentioned locations.
Additionally, each location is marked and becomes clickable on the provided map. Clicking a waypoint marker reveals photos taken at points of interest at the selected location by integrating geolaction with the Flickr API. In addition to these suggested points of interest, Wanderer also provides suggestions for places to lodge and places to eat near each waypoint, again from the integration with Google Map's API.
